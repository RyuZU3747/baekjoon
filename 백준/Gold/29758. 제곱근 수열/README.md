### 문제 설명

<p>양의 정수 $N$과 $L$에 대해, 다음 조건을 만족하는 수열 $A_1, A_2, \cdots, A_L$을 <strong>제곱근 수열</strong>이라고 한다.</p>

<ul>
	<li>$A_1 = N$, $A_L = 1$</li>
	<li>모든 $1 \le i < L$에 대해, $\sqrt{A_i} > A_{i+1}$이다.</li>
	<li>모든 항은 양의 정수이다.</li>
</ul>

<p>$N$과 $L$이 주어질 때, 제곱근 수열의 개수를 구하시오.</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1 \le T \le 100)$</p>

<p>두 번째 줄부터 $T$개의 줄에 걸쳐, 양의 정수 $N$과 $L$이 공백으로 구분되어 주어진다. $(1 \le N \le 70\,000;$ $1 \le L \le 500)$</p>

<p>주어지는 모든 $N$의 합은 $70\,000$ 이하이다.</p>

### 출력 

 <p>$T$개의 줄에 걸쳐, 각각의 $N$과 $L$에 대해 제곱근 수열의 개수를 출력한다.</p>
